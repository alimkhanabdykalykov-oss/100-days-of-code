# 100-days-of-code
100-day journey to sharpen my skills in computational physics, engineering, and mathematics (low probability chemistry related). 
My primary stack is Python, MATLAB, and possibly the Wolfram Language. 
I’ll focusing on deep, complex projects, so I won’t necessarily be finishing a new project every single day; instead, I’ll be iterating on substantial work that takes multiple days to get right.

My personal ground rules:

Originality: I am writing every single line of code myself. Zero vibe-coding.

AI Policy: I only use AI for educational concepts, brainstorming, and debugging logic. The actual implementation is always my own.

Accountability: I’ll be tracking my progress in the log below. If I miss a day, I’ll own it and mark it as a miss.

Technical Focus:

Physics (Quantum mechanics, thermodynamics, waves etc.)

Engineering (Heat transfer, systems modeling etc.)

Math (Analytical derivations, transforms, statistics etc.)

Might do quantum chemistry simulations and calculations (but not so interested in that)

This repo is the raw, unfiltered record of the work.
